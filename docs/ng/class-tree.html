  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • exceptions.StandardError: Base class for all standard Python exceptions that do not represent interpreter exiting.
          • ng.excepts.NGError: Base class for exceptions in this package.
            • ng.database.DatabaseIntegerityError: Error that is raised when failed to execute operations in Database because of data integerity constraints such as UNIQUE keys.
            • ng.excepts.HttpError: Error that is raised when an HTTP error code is required to send.
              • ng.database.DatabaseAccessError: Error that is raised when failed to access database.
              • ng.excepts.FileLocateError: Error that is raised when failed to locate a file.
              • ng.excepts.FileReadError: Error that is raised when failed to read a file.
              • ng.excepts.FileWriteError: Error that is raised when failed to write a file.
              • ng.models.ModelError: Error that is raised when failed to process models.
              • ng.views.TemplateFormatError: Error that is raised when a template contains illegal format.
            • ng._template_loader.TemplateEvalError: Error that is raised when unable to evaluate template string.
            • ng._template_loader.TemplateSplitError: Error that is raised when unable to split template string.
    • ng.database.Database: Abstract class that defines the API of databases.
      • ng.database.MySQLDatabase: Database that connects to MySQL.
    • ng.http.HttpCookie: Class that represents HTTP cookies.
    • ng.http.HttpRequest: HTTP request class that stores information of the client request.
    • ng.http.HttpResponse: HTTP response class that stores information of server response.
      • ng.http.HttpErrorResponse: The HTTP response that indicates an HTTP error.
      • ng.http.HttpRedirectResponse: The HTTP response that redirects the request to another location.
    • ng.http.HttpSession: Abstract class that defines API of HTTP session.
      • ng.http.DatabaseSession: Http session implemented with database.
    • ng.views.View: Empty view.
      • ng.views.BinaryDataView: View that send a binary file to client.
      • ng.views.ImageView: View that displays an image.
      • ng.views.StaticView: View that displays the content of a static html file.
      • ng.views.TemplateView: View that displays html page loaded from a template.
    • dict: dict() -> new empty dictionary dict(mapping) -> new dictionary initialized from a mapping object's (key, value) pairs dict(iterable) -> new dictionary initialized as if via: d = {} for k, v in iterable: d[k] = v dict(**kwargs) -> new dictionary initialized with the name=value pairs in the keyword argument list.
      • ng.models.DatabaseModel: Abstract class that defines models stored in database.
        • ng.models.Account: Model that stores information of user accounts.
        • ng.models.Avatar: Model that stores information of avatars.
        • ng.models.Email: Model that stores information of email addresses.
        • ng.models.Session: Model that stores data and attributes of HTTP sessions.
    • type: type(object) -> the object's type type(name, bases, dict) -> a new type
      • abc.ABCMeta: Metaclass for defining Abstract Base Classes (ABCs).
